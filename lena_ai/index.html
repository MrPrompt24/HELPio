    Klucz API Groq

    Wklej swój klucz z console.groq.com. Klucz zapisujemy lokalnie w przeglądarce (localStorage).

    Jak używać?

    MrPrompt AI Connect to interaktywny czat AI oparty na modelach Groq, działający w całości w Twojej przeglądarce. Klucz API jest przechowywany lokalnie, a cała komunikacja przebiega bezpośrednio i bezpiecznie przez HTTPS.

    Klucz API

    • Otwórz panel Klucz API i wklej swój klucz gsk_….
    • Klucz zapisuje się lokalnie w localStorage – nie opuszcza Twojej przeglądarki.
    • Przycisk „Pobierz klucz” prowadzi do strony console.groq.com, gdzie uzyskasz darmowy dostęp.

    Profile Asystenta

    • Model – wybierz model AI (np. Llama 3.3, Mixtral, Gemma). Listę możesz odświeżyć klikając ikonę .
    • Temperatura – suwak od 0 do 1 (niższa = bardziej precyzyjne, wyższa = bardziej kreatywne odpowiedzi).
    • System prompt – tekst określający osobowość i rolę asystenta. Domyślnie: „Jesteś pomocnym, zwięzłym asystentem. Odpowiadaj po polsku.”
    • Styl interfejsu – do wyboru: Ciemny (bazowy), Jasny, Glassmorphism.
    • Eksport/Import historii – możesz zapisać wszystkie rozmowy do pliku JSON lub wczytać je ponownie.

    Korzystanie z czatu

    • Wysyłanie wiadomości – Enter = wyślij, Shift+Enter = nowa linia.
    • Zatrzymywanie odpowiedzi – przycisk „Stop” zatrzymuje generowanie odpowiedzi.
    • Kopiowanie – przy każdej odpowiedzi AI znajduje się przycisk „Kopiuj”.
    • Wyczyść czat – usuwa całą rozmowę z bieżącej sesji.

    Historia rozmów

    • Rozmowy są zapisywane w Twojej przeglądarce (localStorage).
    • Dzięki opcji Eksport/Import możesz przenosić historię pomiędzy urządzeniami.

    Wskazówki

    • Możesz w każdej chwili zmienić model lub system prompt bez utraty historii rozmowy.
    • Wartość temperatury 0.2–0.4 daje odpowiedzi bardziej rzeczowe, a 0.7+ – bardziej kreatywne.
